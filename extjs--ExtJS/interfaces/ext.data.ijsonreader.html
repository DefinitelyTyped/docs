[Method] Relays selected events from the specified Observable as if the events were fired by this

Parameters

  • origin?: any optional

    Object The Observable whose events this object is to relay.

  • events?: Array<string> optional

    String[] Array of event names to relay.

  • prefix?: string optional

    String A common prefix to prepend to the event names. For example: this.relayEvents(this.getStore(), ['load', 'clear'], 'store'); Now the grid will forward 'load' and 'clear' events of store as 'storeload' and 'storeclear'.
